About The Project

The Travel Planner is an interactive web application designed to simplify the complex process of organizing trips. Our mission is to provide users with a seamless platform where they can discover breathtaking destinations, manage their budgets, and create memories that last a lifetime.

Current Development Stage

At this stage of development (Assignment 1 - Part 2), we have successfully implemented:

  • Dynamic Routing: Utilizing Express.js to handle multiple pages and navigation flawlessly.
  • Backend Processing: A robust server-side logic that captures user inquiries via a custom-built contact form.
  • Data Management: Implementation of file system modules to securely store user feedback in JSON format for future analysis.
  • Interactive UI: A responsive design featuring destination filtering and a personalized "Favorites" system to enhance user engagement.

Future Goals

Moving forward, our team is focused on integrating budget calculators, real-time weather updates for destinations, and a comprehensive search engine to further personalize the travel planning experience.
